[영어 이미지를 한국어 음성파일로 변환]

AWS 사용 기능 : Lambda -A , Lambda -B , S3 -A , S3-B, TexTract, Translate. Polly

이 기능은 안드로이드 앱을 통해 이미지를 전송(영어가 쓰여진) 하면,

그 이미지가 연결된 AWS S3-A 버킷에 업로드가 됩니다.

S3 버킷에 이미지가 업로드가 되면, 이미지를 업로드 받은 S3와 연결된 AWS Lambda - A 에서

특정 동작을 수행합니다.

그 동작은 S3 - A에 업로드된 파일이 이미지일 경우, 
AWS TexTract를 사용해서 이미지에서 OCR 추출을 진행합니다. (문자추출)

문자추출이 완료되면 해당 내용을 텍스트 파일로 만듭니다.
(영어 이미지를 영어 텍스트로 변환)

영어로 변환된 텍스트를 AWS Translate를 사용하여 영어를 한국어로 변환한 후

한국어로 번역된 내용을 텍스트 파일로 생성한 뒤, 새로운 S3 - B에 전송합니다.

S3-B 에 텍스트가 업로드 되는 이벤트가 발생하면, Lambda-B가 동작합니다.

그 동작은 S3-B에 업로드되는 파일이 텍스트일 경우,
AWS Polly 를 사용하여 텍스트를 음성으로 변환합니다. 

음성으로 변환된 파일은 MP3 File로 변환되어 S3에 저장됩니다.

후에 이 MP3 File을 AI Speaker Alexa를 이용해서 Lambda를 통해 음성파일을 불러오고

그 음성파일이 Alexa에서 들리도록 실행할 수 있습니다.
